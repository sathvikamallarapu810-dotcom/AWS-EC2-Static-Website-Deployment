# AWS-EC2-Static-Website-Deployment
Deploying a static website on AWS EC2 using Apache web server
## Project Overview
This project demonstrates how I deployed a static website on an Amazon EC2 instance.

## AWS Services Used
- Amazon EC2
- Security Groups
- Linux Server
- Apache Web Server

## Architecture
User → Internet → EC2 Instance → Apache Web Server → Static Website

## Steps Performed
1. Created an EC2 instance
2. Connected to EC2 using SSH
3. Installed Apache Web Server
4. Uploaded website files
5. Configured security group rules
6. Accessed website using public IP

## Skills Learned
- EC2 instance management
- Linux commands
- Web server configuration
- AWS networking basics
